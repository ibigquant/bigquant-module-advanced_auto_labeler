## 如何参与修改

一个模块可以有多个版本，每一个版本部署使用后，代码将不可修改。如果需要更新模块，请创建一个新的版本。

1. fork项目
2. 拷贝最新一个版本，来创建一个新的版本号
3. 在新版本里，修改代码
4. 发起pull request
